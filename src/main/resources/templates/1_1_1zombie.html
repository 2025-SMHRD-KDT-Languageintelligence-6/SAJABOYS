1. 🎯 게임 개요

“좀비 아포칼립스”는 생존자와 좀비로 나뉘어 진행되는 추격형 야외 게임입니다. 생존자는 미션을 수행해 탈출을 노리고, 좀비는 생존자를 감염시키는 것이 목표예요. 모바일 기기를 통해 역할 배정미션·아이템감염 추적실시간 알림을 관리합니다.

인원 20명 이상(최대 100명)
시간 30–60분
장소 공원, 캠핑장, 운동장 등
장비 참가자 스마트폰, QR 미션 스탠드, 운영자용 태블릿/노트북

2. 🧑‍🤝‍🧑 역할 및 모바일 기능

역할 배정

게임 시작 전 모바일 앱/웹에서 자동 배정됩니다.

복장 표기

모바일에서 자신의 역할을 확인한 뒤, 현장에서 조끼/팔찌/스티커로 표시합니다.

  • 생존자 미션 수행, 아이템 사용, 탈출 시도
  • 좀비 생존자 추격, 접촉 감염 등록
  • 운영자 실시간 방송, 페널티 처리, 수동 아이템 지급
  • 관람객/NPC QR 제공자 또는 NPC로 참여 가능

3. 🗺️ 게임 구역 및 모바일 연동

구역 설정

  • 시작 지점
  • 미션 지점 (QR 코드 부착)
  • 탈출 지점
  • 감염 구역 (좀비 부활 장소)

모바일 기능

  • 지도 기반 위치 안내
  • 미션 지점 근처 도달 시 푸시 알림
  • 탈출 가능 여부 실시간 표시

4. 📜 게임 진행 방식

4.1 시작

  1. 참가자는 모바일로 역할 확인 후 지정된 시작 지점으로 이동
  2. 생존자는 미션 지점에서 QR 스캔 후 미션 수행
  3. 좀비는 생존자를 추적하여 감염 시도

4.2 감염 규칙

  • 좀비가 생존자에게 손을 대고 앱에서 “감염 등록” 버튼 클릭
  • 생존자는 앱에서 감염 여부 확인
  • 해독제 보유 시 즉시 해제 가능(앱에서 사용)

4.3 미션 예시 (모바일 연동)

  • QR 스캔 후 퀴즈 풀이
  • AR로 숨겨진 아이템 찾기
  • 사진 인증 미션(앱 업로드)
  • NPC 역할 운영자에게 암호 전달 → 앱에서 확인

5. 🎲 아이템 시스템

  • 아이템은 미션 성공 시 자동 지급 또는 운영자 수동 지급
  • 사용 시 앱에서 효과 발동 및 지속 시간 표시
예시 아이템 보기
  • 해독제 — 감염 즉시 해제(재사용 대기시간 적용 가능)
  • 방어막 — 일정 시간 감염 무효
  • 가속화 — 일정 시간 스태미나/이동속도 버프(룰 한정)

6. ⚖️ 페널티 및 공정성

  • 금지 구역 접근 시 앱에서 경고 및 이동 제한
  • 과격한 접촉 시 운영자 경고 및 퇴장 처리
  • 아이템 중복 사용 시 앱에서 자동 무효화
안전이 최우선! 접촉은 손끝으로 가볍게, 넘어짐·부딪힘 위험 행동 금지.

7. 🧠 전략 팁

  • 생존자: 팀을 나눠 미션 분담 후 탈출 지점 합류
  • 좀비: 협력 포위 전략으로 동선 차단
  • 해독제: 아끼지 말고 즉시 사용
  • 운영자: “좀비 강화 시작!” 등 실시간 방송 메시지로 페이싱 조율

8. 🎉 축제 연계 요소

  • 코스튬 콘테스트: 모바일 참가 신청 & 투표
  • 포토 미션: 인증샷 업로드 → 베스트샷 선정
  • 실시간 랭킹: 생존 시간·감염 수 등 앱에서 표시
  • 관람객 참여: QR 미션 제공자 또는 NPC로 참여 가능

🧾 부록A. QR·GPS·카메라·NFC/비콘

1) QR코드 활용

  • 미션 인증: QR 스캔 시 퀴즈·퍼즐·사진 미션 자동 시작
  • 아이템 획득: 스캔 시 해독제·방어막 등 지급
  • NPC 시나리오: 스캔 시 가상 대화/암호 진행
  • 감염 등록: 감염 후 QR 스캔으로 상태 업데이트

운영 팁: QR은 미션 장소·운영본부·아이템 박스에 부착. 스캔 → 미션/지급 자동 트리거.

2) GPS 위치 기반

  • 도달 확인: 특정 좌표 도달 시 미션 자동 시작
  • 탈출 판정: 탈출 지점 GPS 도달 시 승리 처리
  • 근접 경고: 좀비가 일정 거리 내 접근 시 생존자 알림
  • 금지 구역: 출입 시 경고 또는 페널티 부여

권장 설정: GPS 정확도 ±10m, 지도 UI로 참가자 위치 실시간 표기

3) 모바일 카메라

  • 사진 인증: 특정 배경/장소 인증 업로드
  • AR: 숨겨진 아이템·좀비 찾기
  • 포토 콘테스트: 분장 사진 업로드 → 투표
  • 감염 인증샷: 감염 순간 기록

4) NFC/블루투스 비콘(선택)

  • NFC 아이템: 태그 터치로 즉시 지급
  • 비콘 시작: 근접 시 미션 자동 시작
  • 강화 구역: 특정 존 입장 시 좀비 능력 강화
